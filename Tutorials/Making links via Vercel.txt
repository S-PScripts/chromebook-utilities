Making links via Vercel:
Link: https://vercel.com/new/clone?repositoryurl=REPOSITORYURL

1. Replace the filler with the unblocker repo link of your choosing.
2. Go to the link you created.
3. Login with GitHub. Sign up with GitHub if you don't have an account.
-> If you can't login with GitHub, log in with your email. You will have to verify. After verifying, go to the link you created again.
4. In the "Create Git Repository", enter the repository name that will be linked to your unblocker. It can be anything.
5. Click "Create". You will need to confirm it with GitHub.
6. After a few moments, go to the link that you just created. It should be (REPOSITORYNAME).vercel.app.
